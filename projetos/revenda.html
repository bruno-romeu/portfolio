Revenda de Carros

Sistema de revenda de veículos desenvolvido com Django, HTML e CSS.

Tecnologias Utilizadas

  • Django (Python)
  • HTML e CSS
  • PostgreSQL
  • API Gemini (IA)

Funcionalidades

  • Cadastro e exibição de veículos disponíveis para revenda
  • Página de detalhes do veículo com informações completas e imagem
  • Geração automática de descrições e observações usando IA (API Gemini)
  • Layout responsivo e navegação simples
  • Sistema de rotas com Django (URLs e views organizadas)

Desafios e Aprendizados

Esse projeto foi essencial para aprofundar os conhecimentos em Django, principalmente na organização de modelos, views e templates.

Foi uma oportunidade de aplicar na prática o uso de rotas dinâmicas e a conexão com banco de dados. Também trouxe aprendizado prático com integração de APIs externas, usando a API Gemini para gerar descrições automáticas a partir dos dados do veículo.

Foi uma ótima experiência de aplicação de inteligência artificial no contexto web, além de reforçar boas práticas de estruturação de páginas e separação entre lógica e visual.
