Blog

Blog simples feito com PHP e dados estáticos para exibir posts organizados por categorias.

Tecnologias Utilizadas

  • PHP puro
  • HTML e CSS
  • Bootstrap

Funcionalidades

  • Exibição de posts com título, conteúdo e imagem
  • Organização por categorias
  • Layout responsivo e navegação simples

Desafios e Aprendizados

Esse projeto foi importante para entender a lógica de organização de conteúdo em páginas, a separação entre partes reutilizáveis (como header e footer) e a construção de páginas dinâmicas com dados simulados em PHP. Foi um bom exercício de estrutura de site antes de lidar com banco de dados.
