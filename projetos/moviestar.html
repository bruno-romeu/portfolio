MovieStar

Site de avaliações de filmes com sistema de usuários e notas personalizadas.

Tecnologias Utilizadas

  • PHP orientado a objetos
  • MySQL
  • HTML, CSS e Bootstrap

Funcionalidades

  • Cadastro e login de usuários
  • Cadastro, edição e exclusão de filmes
  • Sistema de avaliação com estrelas
  • Criação de reviews e comentários
  • Exibição de filmes por categorias

Desafios e Aprendizados

Durante o desenvolvimento do MovieStar, aprofundei meus conhecimentos em POO com PHP, sessions, autenticação de usuários e relacionamentos entre tabelas no banco de dados. Aprendi também a lidar com uploads de imagens, validações de segurança e a exibir dados dinâmicos de forma organizada.
