Agenda

CRUD de contatos desenvolvido com PHP e MySQL.

Tecnologias Utilizadas

  • PHP puro
  • MySQL
  • HTML, CSS e Bootstrap

Funcionalidades

  • Cadastro, edição e exclusão de contatos
  • Listagem com paginação
  • Validação de dados no formulário

Desafios e Aprendizados

Durante o desenvolvimento, aprimorei minha lógica com PHP, entendi melhor como estruturar operações CRUD e também como separar partes do layout com reaproveitamento de código (header, footer, etc).
